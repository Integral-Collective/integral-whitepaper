# Contributing to Integral

Thank you for your interest in contributing to the Integral build.

This project gates participation through a deliberate application 
process. Before you can open issues, comment, or submit pull requests 
you need to be an approved contributor.

**To apply:** Visit integralcollective.io/application.html

The application takes approximately 15 minutes and is reviewed 
personally. If there is a good fit you will hear back within one week 
with an invitation to join the GitHub organization and Discord community.

If you have already been approved and are having trouble with access, 
contact us via the website.
